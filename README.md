# Starbucks Clone
### 배포 링크
https://ramstarbucks.netlify.app/

* 스타벅스 페이지를 Html, Css, JavaScript로 클론 코딩했습니다.
* 메인페이지와 로그인 페이지를 구현했습니다
* 시맨틱 요소를 사용하여 레이아웃을 구분하고 cdnjs를 사용했습니다
